# Screen: Life Happens Menu
**ID:** PLAN-02
**Parent Flow:** Plan Tab -> "Life Happens" Button

## 1. Purpose
To allow the user to report interruptions (Illness, Travel, Missed Workout) and trigger the Adaptive Engine.

## 2. Layout & Structure
*   **Type:** Bottom Sheet Menu.

## 3. UI Components

### 3.1. Options List
*   **Option 1:** "I'm Sick"
    *   **Icon:** Thermometer / Bacteria.
    *   **Action:** Opens Illness Flow (Neck Check).
*   **Option 2:** "I'm Traveling"
    *   **Icon:** Plane.
    *   **Action:** Opens Date Picker for travel dates.
*   **Option 3:** "I Missed a Workout"
    *   **Icon:** Calendar X.
    *   **Action:** Opens Reschedule Flow (`PLAN-03`).
*   **Option 4:** "I have an Injury"
    *   **Icon:** Bandage.
    *   **Action:** Opens Injury Report.

## 4. Dynamic Data
*   None (Static Menu).

## 5. Design System References
*   **Icons:** Large, friendly icons.
*   **Tone:** Empathetic. "Don't worry, we'll adjust."
